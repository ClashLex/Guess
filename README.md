# Guess

## Project Description
Guess is an interactive guessing game that challenges players to guess a number within a certain range. It incorporates both single-player and multiplayer modes.

## Features
- Single-player and multiplayer modes
- User-friendly interface
- High score tracking
- Adaptive difficulty levels

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/ClashLex/Guess.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Guess
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

## Usage
To start the game, run the following command:
```bash
npm start
```
Follow the on-screen instructions to begin playing!
